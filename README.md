ES (Español)
=============

INSTRUCCIONES DE INSTALACIÓN

Opción A: Windows (Ejecutable .exe)
1. Descomprime el archivo ZIP completamente (Clic derecho -> Extraer todo).
2. Abre la carpeta extraída.
3. Haz doble clic en "Memorama_Electronico.exe" para jugar.

   NOTA: 
   Si aparece una ventana azul diciendo "Windows protegió su PC":
   1. Haz clic en el texto subrayado "Más información".
   2. Haz clic en el botón "Ejecutar de todas formas".

Opción B: Linux / macOS / Código Fuente
Si deseas ejecutar el juego desde el código original:

1. Asegúrate de tener Python 3 instalado en tu sistema.
2. Abre una terminal en la carpeta del juego.
3. Instala las dependencias necesarias ejecutando:
   pip install pygame
4. Inicia el juego con el siguiente comando:
   python run.py

ENG (English)
=============

INSTALLATION INSTRUCTIONS

Option A: Windows (Executable .exe)
1. Unzip the file completely (Right-click -> Extract All).
2. Open the extracted folder.
3. Double-click "Memorama_Electronico.exe" to play.

   NOTE:
   If a blue window appears saying "Windows protected your PC":
   1. Click on "More info".
   2. Click the "Run anyway" button.

Option B: Linux / macOS / Source Code
If you want to run the game from the source code:

1. Ensure you have Python 3 installed on your system.
2. Open a terminal inside the game folder.
3. Install the required dependencies by running:
   pip install pygame
4. Start the game with the command:
   python run.py
